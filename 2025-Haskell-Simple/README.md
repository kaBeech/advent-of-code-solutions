## Executable

Run all commands from the root directory (i.e. the one that contains `src/`).

To run tests for all valid Days: `cabal test`

Make sure to add your puzzle inputs to `data/Day${DAYNUMBER}/challenge_input.dat`
before running any of the solution commands below!

To get solutions for all valid Days: `cabal run`

To get solutions for a specific Day: `cabal run exes -- ${DAYNUMBER}`
